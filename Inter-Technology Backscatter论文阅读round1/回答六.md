## 回答六

## 1. FPGA 平台

**FPGA** 全称是 **Field-Programmable Gate Array**,中文常叫"现场可编程门阵列"。

它可以理解成一种"可反复改电路逻辑的芯片"。

普通芯片做好以后功能基本固定,而 FPGA 可以通过下载配置文件,改变内部电路连接。所以研究人员常用 FPGA 做硬件原型,因为它:

- 比真正流片做芯片便宜;
- 修改设计方便;
- 可以验证算法和电路是否真的可行;
- 适合做通信、信号处理、硬件加速等实验。

在这段话里,"在 FPGA 平台上构建反向散射硬件原型"的意思是:  
他们还没有一开始就做真正的专用芯片,而是先用 FPGA 搭了一个实验版本。

## 2. Cadence 和 Synopsys

这里原文写的 **Synopsis** 很可能是 **Synopsys**,它是一家著名的电子设计自动化公司。

**Cadence** 和 **Synopsys** 都是芯片设计领域常用的软件工具厂商。

## 3. 集成电路 IC

**集成电路**英文是 **Integrated Circuit**,简称 **IC**,也就是我们常说的"芯片"。

FPGA 是可编程硬件,适合做原型;  
IC/ASIC 是专门为某个功能设计的芯片,适合量产和低功耗。

二者可以这样对比:

| 概念        | 特点          | 在这段话里的作用   |
| --------- | ----------- | ---------- |
| FPGA      | 可编程、灵活、方便实验 | 用来搭建硬件原型   |
| IC / ASIC | 定制化、效率高、功耗低 | 用来估计最终芯片能耗 |
|           |             |            |

所以研究路线大概是:

> 先用 FPGA 验证功能可行,再用 Cadence/Synopsys 设计 IC 版本估算功耗。
